**Toko Alat Musik Web Service**

**Deskripsi Proyek**
Toko Alat Musik Web Service adalah sistem backend berbasis REST API yang dikembangkan menggunakan Laravel.  
Sistem ini menyediakan layanan pengelolaan data produk alat musik, kategori, pelanggan, pesanan (orders), serta autentikasi pengguna menggunakan JWT.

Proyek ini dibuat untuk memenuhi tugas UAS Pemrograman Web Service.

---

## ⚙️ Teknologi yang Digunakan
- Laravel
- MySQL
- JWT Authentication
- Postman (untuk pengujian API)

---

**Cara Menjalankan Sistem:**
Clone repository
Masuk ke folder project
Install dependency (Composer)
Copy & setting file .env
Generate key (Laravel)
Migrasi database
Jalankan server
